# Web Geliştirme JS Örneği

Bu proje, ReactJS ve Vite kullanarak geliştirilmiş bir Harcama Takipçisi uygulamasıdır. Kullanıcılar gelir ve giderlerini ekleyip takip edebilir.

## Kurulum ve Çalıştırma

### Ön Gereksinimler

- Node.js (versiyon 14 veya üzeri)
- npm veya yarn

### Adımlar

1. **Proje bağımlılıklarını yükleyin:**

   ```
   npm install
   ```

2. **Geliştirme sunucusunu başlatın:**

   ```
   npm run dev
   ```

Artık uygulamanız [http://localhost:5173](http://localhost:5173) adresinde çalışıyor olmalı!
